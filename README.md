# Text_to_Speech

A simple and user-friendly Text to Speech (TTS) Converter desktop application built with Python and Tkinter.
This project allows users to enter text and convert it into speech using Python’s text-to-speech libraries.

🚀 Features:
Convert text into speech instantly,
Simple and clean GUI using Tkinter,
Adjustable speech rate,
Lightweight and beginner-friendly project.

🛠️ Technologies Used :
Python,
Tkinter.
